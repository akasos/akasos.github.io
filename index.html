Work Experience

Hill Air Force Base

I currently work at HAFB as a Software Engineer. I work on a team that develops and maintains avionics software for the F-16. Work life involves coding in C and C++ and writing an occasional Bash or Perl script here or there. On the daily, we do regression testing, code functional testing, document re-writes, code changes and everything else that is needed to keep the pilots happy and flying.

Fedex Express

At FedEx I did what you probably think I did. I did pickups and deliveries. During the evenings I would help load out-bound packages into cans which would then to be sent off to the airport for further processing. I was also a Dangerous Good's Specialist which involved me making sure no hazardous materials were unfit for travel. On top of all that I drove like a champ and was accident-free for the entire 3 year's I worked there.
